Basketball Tickets

Web Assistant

This project involved redesigning the Basketball Tickets page for Hope College to create a clearer, more intuitive experience for fans navigating game information and ticket options.

Using HTML, CSS, and JavaScript, I reworked the overall layout and content flow, introducing clear visual indicators to distinguish men’s and women’s games. This helped reduce confusion and allowed users to quickly find the information most relevant to them.

A key UX feature automatically moves past games into a dedicated history tab, keeping the primary view focused on upcoming events. Implementing this logic required thoughtful handling of game data and reinforced my skills in writing user-centered JavaScript interactions.

The page also includes interactive search and organization features, making it easier to explore games across devices. Through this project, I strengthened my ability to design responsive, accessible interfaces that balance visual clarity with functional performance.
